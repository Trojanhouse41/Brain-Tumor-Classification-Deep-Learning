#  Brain Tumor Classification Using Deep Learning

## Overview
This project develops a deep learning-based image classification system capable of identifying multiple brain tumor classes from MRI scans. The system uses Convolutional Neural Networks (CNNs) and Transfer Learning to achieve high diagnostic accuracy.

## Project Objectives
- Develop a CNN model to classify MRI scans into 4 classes: Glioma, Meningioma, Pituitary Tumor, and No Tumor.
- Achieve a validation accuracy of at least 80%.
- Deploy the model for mobile and web applications.

##  Results
- **Final Validation Accuracy:** 86.28% 
- **Optimization Techniques:** Transfer Learning (MobileNetV2), Data Augmentation, Dropout, Batch Normalization, and Early Stopping.

##  Tech Stack
- **Language:** Python
- **Framework:** TensorFlow / Keras
- **Environment:** Google Colab (GPU)
- **Deployment:** TensorFlow Lite (.tflite), HDF5 (.h5)

##  Repository Contents
- `Brain_Tumor_Classification.ipynb`: The complete Google Colab notebook containing data preprocessing, model building, training, and evaluation.
- `Project_Report.pdf`: The comprehensive PDF report detailing the methodology, results, and deployment strategy.
- `deployment/`: Folder containing the saved models and API code.

## How to Run
1. Open the `.ipynb` file in Google Colab.
2. Ensure you are using a GPU runtime (Runtime -> Change runtime type -> T4 GPU).
3. Run the cells sequentially from top to bottom.
